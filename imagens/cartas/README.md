# Pasta de Sprites das Cartas 🎴

Coloque as sprites ilustradas das cartas nesta pasta.

O jogo foi configurado para buscar os arquivos automaticamente usando o nome da carta em formato minúsculo e sem acentos, com espaços substituídos por sublinhados (`_`).

### Exemplos de Nomes de Arquivos:

*   **Espada** ➔ `espada.png`
*   **Escudo** ➔ `escudo.png`
*   **Poção** ➔ `pocao.png`
*   **Curativo** ➔ `curativo.png`
*   **Coringa de Bronze** ➔ `coringa_de_bronze.png`
*   **Ímpar Sagrado** ➔ `impar_sagrado.png`
*   **Golpe G.** ➔ `golpe_g.png`
*   **Adaga Tóxica** ➔ `adaga_toxica.png`
*   **ULTRA** ➔ `ultra.png`
*   **REI** ➔ `rei.png`
*   **CORINGA** ➔ `coringa.png`
*   **Tempestade Rúnica** ➔ `tempestade_runica.png`

### Cartas Básicas (Números de 1 a 10):
As cartas numéricas básicas também podem ter sprites personalizadas!
*   **"1"** ➔ `1.png`
*   **"2"** ➔ `2.png`
*   ...
*   **"10"** ➔ `10.png`

### Recomendações:
*   Fundo transparente (`.png`).
*   Proporção sugerida: `300x450px` ou similar (vertical).
*   Se o arquivo não existir na pasta, o jogo continuará usando o ícone SVG original automaticamente (fallback automático)!
